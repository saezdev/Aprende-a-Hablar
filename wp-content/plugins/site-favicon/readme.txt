== Description ==

Add a favicon.

Set the favicon URL under *Appearance > Customize > Site Identity > Site Favicon*.

*This plugin is especially useful if you'd like to set a Site Icon, but also need to set a unique favicon separate from it.*

== Installation ==

**Automatic**

* From your WordPress Admin, navigate to: **Plugins** > **Add New**
* Search for: **Site Favicon**
* Install it
* Activate it

**Manual**

* Download
* Unzip
* Upload to /plugins/ folder
* Activate
